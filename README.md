- 파일 이름은 자유, 다만 commit 형식 반드시 준수
## commit 메세지: [문제 출처(플랫폼)] 문제이름 / 문제난이도
- description: 문제 주소 (option)
- 터미널에서 작성법:
`git commit -m "[BOJ] Hello World / 브론즈5 / 0000" -m "https://www.acmicpc.net/problem/2557"`
- 플랫폼 작성법 통일:
  - [BOJ] - 백준
  - [PGS] - 프로그래머스
  - [LTC] - 리트코드
  - [CFS] - 코드포스
